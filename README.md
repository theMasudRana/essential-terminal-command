# essential-terminal-command


## Commands for terminal(Ubuntu)
System update.
```
sudo apt-get update
```
Clear terminal screen.
```
ctrl + l
```
Present working directory
```
pwd
```
Creating a folder
```
mkdir FolderName
```
Creating a file
```
touch FileName
```

## Git commands
Initialize git.
```
git init
```
